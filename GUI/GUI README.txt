- GUI can be installed as an app into MATLAB using the "RadarDetectionofSmallBoatsatSea.mlappinstall" file

- GUI can be edited in MATLAB's App Designer using the "RadarDetectionofSmallBoatsatSea.mlapp" file

- GUI will only run after it has first been run within App Designer; thereafter the "RadarDetectionofSmallBoatsatSea.mlapp" file
  can be used to access it directly, even outside of App Designer
